# Saxon Genitive ('s)

<Typography variant="p" className="text-center">
  {`Se utiliza para indicar posesión o relación entre personas y cosas. Generalmente se usa con personas y animales.`}
</Typography>

<br/><br/>

<GrammarExample 
  structure="Possessor + 's + Object"
  examples={[
    { type: "Singular", text: "Wendy's car is red.", translation: "El carro de Wendy es rojo." },
    { type: "Animal", text: "The dog's house.", translation: "La casa del perro." },
    { type: "Plural (ends in S)", text: "My parents' house.", translation: "La casa de mis padres." },
    { type: "Irregular Plural", text: "The children's toys.", translation: "Los juguetes de los niños." }
  ]}
/>

<Alert variant="warn" title="Regla de oro" >
  Si el nombre termina en 's', solo pones el apóstrofo: **Carlos' car**.
  Si es plural y termina en 's', también solo el apóstrofo: **The sisters' room**.
</Alert>

## ¿Cuándo NO usarlo?
Para objetos inanimados, se suele usar la preposición **"of"**:
