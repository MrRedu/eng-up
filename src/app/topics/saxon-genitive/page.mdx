# Saxon Genitive ('s)

<Typography variant="p" className="text-center">
  {`Se utiliza para indicar posesión o relación entre personas y cosas. Generalmente se usa con personas y animales.`}
</Typography>

<Alert variant="warn" title="Regla de oro" >
  Si el nombre termina en 's', solo pones el apóstrofo: **Carlos' car**.
  Si es plural y termina en 's', también solo el apóstrofo: **The sisters' room**.
</Alert>

## ¿Cuándo NO usarlo?
Para objetos inanimados, se suele usar la preposición **"of"**:
