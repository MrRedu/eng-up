## 3. ¿Cuándo usar "A lot of"?

<Typography variant="p">
  {`En oraciones **afirmativas**, es mucho más natural usar **a lot of** o **lots of** tanto para contables como para incontables.`}
</Typography>

<Alert variant="info" title="Regla de oro">
  - **Many / Much**: Se usan principalmente en oraciones **negativas** e **interrogativas**. <br/>
  - **A lot of / Lots of**: Se usan principalmente en oraciones **afirmativas**.
</Alert>
