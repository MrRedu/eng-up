## 1. Can (Poder)

Es el modal más versátil. Lo usamos principalmente en el presente para tres situaciones clave:

<br />

export const CAN_USES = [

  { use: 'Posibilidad real', example: 'It can be very cold in winter.', translation: 'Puede hacer mucho frío en invierno.' },

## 2. Could (Podía / Podría)

Se utiliza como el pasado de "can" o como una forma más educada y condicional.

<br />

export const COULD_USES = [
  { use: 'Habilidad en el pasado', example: 'When I was young, I could run fast.', translation: 'Cuando era joven, podía correr rápido.' },
  { use: 'Petición cortés', example: 'Could you open the window, please?', translation: '¿Podría abrir la ventana, por favor?' },
  { use: 'Sugerencia o posibilidad', example: 'We could go to the cinema tonight.', translation: 'Podríamos ir al cine esta noche.' },

## 3. Must (Deber)

Expresa una obligación ineludible o una conclusión lógica muy clara.

<br />

export const MUST_USES = [
  { use: 'Obligación fuerte', example: 'You must wear a uniform at school.', translation: 'Debes usar uniforme en la escuela.' },
  { use: 'Deducción lógica (Certeza)', example: 'You have worked all day. You must be tired.', translation: 'Has trabajado todo el día. Debes estar cansado.' },
  { use: 'Prohibición (Mustn\'t)', example: 'You mustn\'t smoke here.', translation: 'No debes fumar aquí. (Prohibido)' },
];

<Alert variant="info" title="Diferencia Crucial: Mustn't vs. Don't have to">
  **Mustn't** significa que algo está prohibido o es una muy mala idea.  
  **Don't have to** significa que no es necesario, pero tienes la opción de hacerlo.
</Alert>

## Estructura General

Todos los modales comparten la misma estructura gramatical simple:

<br />

<GrammarExample 
  structure={[

    { label: "Interrogative", value: "Modal + Subject + Verb (base form)?" }
  ]}
  examples={[
    { type: "Affirmative", text: "He can play the guitar.", translation: "Él puede tocar la guitarra." },

<Alert variant="destructive" title="Importante">
    Nunca uses **"to"** después de los modales. Decir **"I can to swim"** es un error común que debes evitar.
</Alert>
