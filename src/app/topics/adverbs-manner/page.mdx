## 1. Formación de los Adverbios de Modo

La mayoría se forman añadiendo **-ly** al adjetivo correspondiente:

<br />

export const MANNER_FORMATION = [
  { adjective: 'Quick', adverb: 'Quickly', translation: 'Rápido / Rápidamente' },
  { adjective: 'Careful', adverb: 'Carefully', translation: 'Cuidadoso / Cuidadosamente' },
  { adjective: 'Quiet', adverb: 'Quietly', translation: 'Silencioso / Silenciosamente' },
  { adjective: 'Bad', adverb: 'Badly', translation: 'Malo / Malamente' },
];

## 2. Adverbios Irregulares

Algunos adverbios no siguen la regla del **-ly**:

<br />

export const IRREGULAR_MANNER = [
  { adjective: 'Good', adverb: 'Well', translation: 'Bien' },
  { adjective: 'Fast', adverb: 'Fast', translation: 'Rápido' },
  { adjective: 'Hard', adverb: 'Hard', translation: 'Duro / Fuerte' },
  { adjective: 'Late', adverb: 'Late', translation: 'Tarde' },
];

<br />

<Alert variant="warn" title="Nota Importante">
  Ten cuidado con **"hardly"** y **"lately"**. Aunque parecen los adverbios de mode de *hard* y *late*, tienen significados diferentes:
  
  **Hardly:** Casi no / Apenas. <br/>
  **Lately:** Últimamente.
</Alert>
