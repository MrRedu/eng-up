## 2. Reglas Importantes

<List>
  <ListItem className='flex flex-col md:flex-row items-start gap-2'>

    <Typography className="mt-0!">Siempre van colocados **antes** del <HasTranslation translation="el ente/la cosa">sustantivo</HasTranslation>.</Typography>
  </ListItem>
  <ListItem className='flex flex-col md:flex-row items-start gap-2'>
    <strong>Sin plural:</strong>
    <Typography className="mt-0!">No tienen forma plural. Se dice **my book** y **my books** *(nunca "mys books")*.</Typography>
  </ListItem>
  <ListItem className='flex flex-col md:flex-row items-start gap-2'>
    <strong>Género:</strong>
    <Typography className="mt-0!">Dependen de quién posee el objeto (**His house** = la casa de él / **Her house** = la casa de ella).</Typography>
  </ListItem>
</List>

## 3. Confusiones Comunes

<br/>

export const POSSESSIVE_ADJECTIVES_CONFUSIONS = [
  {
    correct: <>The dog wagged <strong>its</strong> tail.</>,
    correctTranslation: "El perro movió su cola.",
    incorrect: <>The dog wagged <strong>it's</strong> tail.</>
  },
  {
    correct: <><strong>Your</strong> house is beautiful.</>,
    correctTranslation: "Tu casa es hermosa.",
    incorrect: <><strong>You're</strong> house is beautiful.</>
  },
  {
    correct: <>I like <strong>their</strong> car.</>,
    correctTranslation: "Me gusta su carro.",
    incorrect: <>I like <strong>there</strong> car.</>
  },
];

  **Your** (posesivo) vs. **You're** (You are). <br/>
