# Comparative & Superlative Adjectives

<Typography variant="p" className="text-center text-balance">
  {`Usamos los comparativos para contrastar dos cosas y los superlativos para destacar una cosa dentro de un grupo. La regla de oro: la longitud del adjetivo determina qué estructura usar.`}

## 1. Comparativos (Contrastando dos cosas)

Para formar el comparativo, clasificamos los adjetivos según su longitud y terminación:

<br />

export const COMPARATIVE_RULES = [

  { type: 'Adj. terminados en "-y"', rules: <>Cambiar "y" por "-ier" + than</>, examples: 'Happy → Happier than', translation: 'Más feliz que' },

];

<Alert variant="info" title="Recordatorio: CVC">
  Si un adjetivo corto termina en **Consonante + Vocal + Consonante** (como *Big* o *Hot*), debemos duplicar la última consonante antes de añadir el sufijo: **Big → Bigger**, **Hot → Hotter**.
</Alert>

## 2. Superlativos (El máximo nivel)

El superlativo siempre requiere el artículo **"the"**.

<br />

export const SUPERLATIVE_RULES = [
  { type: 'Adjetivos CVC', rules: 'the + (doble consonante) + "-est"', examples: 'The biggest', translation: 'El más grande' },

  { type: 'Adjetivos en "-y"', rules: 'the + adjective + "-iest"', examples: 'The happiest', translation: 'El más feliz' },
  { type: 'Adjetivos Largos', rules: 'the + most + adjective', examples: 'The most expensive', translation: 'El más costoso' },

## 3. Adjetivos Irregulares

Estos rompen todas las reglas. Memorízalos, son fundamentales.

<br />
