## 1. Las Palabras Interrogativas

A continuación se presentan las palabras WH más comunes y el tipo de información que solicitan:

<Alert variant="info" title="Who vs Whom">
  En el inglés moderno, **Who** se usa casi siempre. **Whom** es mucho más formal y se utiliza específicamente cuando la persona es el **objeto** de la oración (a quien le sucede la acción) o después de una preposición (e.g., *To whom...*).
</Alert>

## 2. Estructura de la Pregunta

Para formar preguntas correctamente, la estructura cambia dependiendo del tipo de verbo principal que estemos utilizando.

<br />

### A. Con el "Verb to Be"
Se utiliza para preguntar sobre estados, identidades o ubicaciones.

<br />

<GrammarExample 
  structure={[
    {  value: "WH word + Verb to Be (am/is/are/was/were) + Subject + ...?" },
  ]}
  examples={[
    { type: "Identity", text: "Who is that man?", translation: "¿Quién es ese hombre?" },
    { type: "Location", text: "Where are my shoes?", translation: "¿Dónde están mis zapatos?" },
    { type: "Time", text: "When was the exam?", translation: "¿Cuándo fue el examen?" }
  ]}
/>

<br />
<br />

### B. Con Auxiliares (Acciones)
Se utiliza para preguntar sobre acciones en diferentes tiempos verbales.

<br />

<GrammarExample 
  structure={[
    { value: "WH word + Aux verb (do/does/did/will) + subject + verb + ...?" },
  ]}
  examples={[
    { type: "Present", text: "What do you do for fun?", translation: "¿Qué haces para divertirte?" },
    { type: "Past", text: "Why did she leave early?", translation: "¿Por qué se fue ella temprano?" },
    { type: "Future", text: "How will they pay for it?", translation: "¿Cómo lo pagarán ellos?" }
  ]}
/>

## 3. Variaciones Avanzadas

<br />

### Combinaciones con "How"
La palabra **How** cambia su significado según el adjetivo o adverbio que la acompañe.

<br />

<GrammarExample 
  structure={[
    { label: "How + Adjetivo", value: "How far / How long / How many / How much" },
    { label: "Pregunta", value: "is it? / does it take? / people were there?" }
  ]}
  examples={[
    { type: "Distance", text: "How far is the airport?", translation: "¿Qué tan lejos está el aeropuerto?" },
    { type: "Duration", text: "How long is the movie?", translation: "¿Cuánto dura la película?" },
    { type: "Quantity", text: "How many apps do you have?", translation: "¿Cuántas apps tienes?" },
    { type: "Price", text: "How much did you pay?", translation: "¿Cuánto pagaste?" }
  ]}
/>

<br />

<Alert variant="warn" title="Preguntas de Sujeto">
  Cuando la palabra WH (**Who** o **What**) actúa como el **sujeto** (la persona o cosa que hace la acción), **no se usa auxiliar** (*do, does, did*).
  
  **Subject Question**: <HasTranslation translation="¿Quién rompió el vaso?">"Who **broke** the glass?"</HasTranslation> (Correcto) <br/>
  *Incorrect*: "Who did break the glass?"
</Alert>
