# Inferiority Comparison

<Typography variant="p" className="text-center text-balance">
  {`No siempre queremos decir que algo es "más". A veces necesitamos expresar que algo es "menos" o que no alcanza el mismo nivel que otra cosa. Para esto usamos la comparación de inferioridad.`}
</Typography>

<br />
<br />

## 1. Less ... than (Menos ... que)

A diferencia de los comparativos de superioridad, la estructura **"less + adjective + than"** se usa igual para adjetivos cortos y largos. Sin embargo, en la práctica, es más común con adjetivos largos.

<br />

<GrammarExample
  structure="Subject + Verb + less + Adjective + than + Complement"
  examples={[
    { type: 'Long Adj', text: "This exercise is less difficult than the last one.", translation: "Este ejercicio es menos difícil que el anterior." },
    { type: 'Short Adj', text: "The soup is less hot than before.", translation: "La sopa está menos caliente que antes. (Gramaticalmente correcto, pero poco común)." }
  ]}
/>

<br />
<br />

## 2. Not as ... as (No tan ... como)

Esta es la forma más natural y frecuente de expresar inferioridad en el inglés hablado. Decir que algo "no es tan bueno como" suele sonar mejor que decir que es "menos bueno".

<br />

<GrammarExample
  structure="Subject + not as + Adjective + as + Comparison"
  examples={[
    { type: 'Affirmative', text: "Your phone is not as fast as mine.", translation: "Tu teléfono no es tan rápido como el mío." },
    { type: 'Negative', text: "Biology is not as interesting as History.", translation: "La biología no es tan interesante como la historia." },
    { type: 'Negative', text: "I am not as tall as my brother.", translation: "No soy tan alto como mi hermano." }
  ]}
/>

<br />
<br />

## 3. The Least (El menos / La menos)

Es el superlativo de inferioridad. Indica que algo tiene el nivel más bajo de una cualidad dentro de un grupo.

<br />

<GrammarExample
  structure="The + least + Adjective"
  examples={[
    { type: 'Superlative', text: "This is the least expensive shirt in the shop.", translation: "Esta es la camisa menos cara de la tienda." },
    { type: 'Superlative', text: "He is the least talkative person I know.", translation: "Él es la persona menos habladora que conozco." }
  ]}
/>

<br />

<Alert variant="info" title="Teacher's Advice">
  Cuando hables, prioriza **"not as... as"**. Suena mucho más fluido y natural que usar **"less"** con adjetivos cortos. Por ejemplo, en lugar de *"It is less cold"*, prefiere decir *"It is not as cold"*.
</Alert>
