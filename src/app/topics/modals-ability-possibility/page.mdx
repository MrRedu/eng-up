## 1. Expresando Habilidad (Ability)

Usamos **Can**, **Could** y **Be able to** dependiendo del tiempo verbal y el contexto.

<br />

export const ABILITY_RULES = [

  { form: 'Be able to', usage: 'Habilidad en cualquier tiempo verbal (especialmente futuro y perfectos).', example: 'I will be able to help you tomorrow.', translation: 'Podré ayudarte mañana.' },

<Alert variant="info" title="Could vs. Was able to">
  Usamos **Could** para una habilidad general en el pasado. Sin embargo, para una acción específica y exitosa en un momento concreto del pasado, es mejor usar **was/were able to** o **managed to**.
  
  *Ejemplo: <HasTranslation translation="Incluso si la puerta estaba cerrada, pude abrirla." >"Even though the door was locked, I **was able to** open it."</HasTranslation>*
</Alert>

## 2. Expresando Posibilidad (Possibility)

Para hablar de cosas que pueden suceder, usamos **May**, **Might** y **Could**.

<br />

export const POSSIBILITY_LEVELS = [
  { modal: 'May', level: 'Probabilidad media/alta (Factual o formal).', example: 'It may rain later.', translation: 'Puede que llueva más tarde.' },
  { modal: 'Might', level: 'Probabilidad baja (Remota o hipotética).', example: 'We might win the lottery.', translation: 'Podríamos ganar la lotería.' },
  { modal: 'Could', level: 'Posibilidad teórica o sugerencia.', example: 'You could be right.', translation: 'Podrías tener razón.' },
];

<br />

<Alert variant="warn" title="Interrogativas con May/Might">
  No solemos usar **May** o **Might** en preguntas para hablar de posibilidad. En su lugar, preferimos **Could** o la expresión **Do you think...?**
  
  *<HasTranslation translation="Correcto" as="strong">Correct</HasTranslation>: <HasTranslation translation="¿Crees que lloverá?" >"Do you think it will rain?"</HasTranslation>* o *<HasTranslation translation="¿Podría ser verdad?" >"Could it be true?"</HasTranslation>*  
  *<HasTranslation translation="Poco común" as="strong">Uncommon</HasTranslation>: <HasTranslation translation="¿Podría llover?" >"May it rain?"</HasTranslation>*
</Alert>
