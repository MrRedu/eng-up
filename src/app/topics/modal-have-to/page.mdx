# Modal: Have to

<Typography variant="p" className="text-center text-balance">

## 1. Reglas y Conjugación

Al igual que en su uso como verbo de posesión, **have to** cambia a **has to** para la tercera persona del singular (**He, She, It**).

<br />

export const HAVE_TO_RULES = [
  { subject: 'I / You / We / They', form: 'have to', example: 'I have to go to work', translation: 'Tengo que ir a trabajar' },
  { subject: 'He / She / It', form: 'has to', example: 'She has to go to work', translation: 'Ella tiene que ir a trabajar' },
];

## 2. Estructuras Gramaticales

Para las formas negativa e interrogativa, **siempre** necesitamos el auxiliar **do / does**.

<br />

<br />

<Alert variant="info" title="Don't have to = No es necesario">
  Es un error común pensar que **don't have to** significa prohibición. En realidad, significa que **no hay obligación** o que algo no es necesario.
</Alert>

<br />

## 3. Have to vs. Must

Aunque ambos expresan obligación, hay un matiz importante:

<br />

export const HAVE_TO_VS_MUST = [
  { modal: 'Must', type: 'Interna (sentimiento personal o consejo fuerte).', example: 'I must call my mom.', translation: 'Tengo que llamar a mi mamá.' },
  { modal: 'Have to', type: 'Externa (regla, leyes, circunstancias).', example: 'I have to wear a seatbelt.', translation: 'Tengo que usar el cinturón de seguridad.' },
];

<Alert variant="warn" title="Peligro: Prohibición">
  Si realmente quieres decir que algo está **prohibido**, usa **Mustn't**.

  *<HasTranslation translation="No debes fumar aquí" as="strong">You mustn't smoke here</HasTranslation>* (Prohibido).<br/>
  *<HasTranslation translation="No tienes que fumar aquí" as="strong">You don't have to smoke here</HasTranslation>* (Puedes hacerlo si quieres, pero no es obligatorio).

</Alert>

<Typography variant="blockquote">
  Recuerda: **Have to** es mucho más común en el habla cotidiana que **Must** para expresar obligaciones generales.
</Typography>
