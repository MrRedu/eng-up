# Future Simple

Se usa para predicciones, promesas, decisiones espontáneas o hechos futuros.

<GrammarExample
  structure="Subject + will + Verb (Base Form) + Complement"
  examples={[
    {
      type: 'Affirmative',
      text: 'I will study English tomorrow.',
      translation: 'Estudiaré inglés mañana.',
    },
    {
      type: 'Negative',
      text: "He will not (won't) go to the party.",
      translation: 'Él no irá a la fiesta.',
    },
    {
      type: 'Interrogative',
      text: 'Will you help me?',
      translation: '¿Me ayudarás?',
    },
    { type: 'Answer (+)', text: 'Yes, I will.', translation: 'Sí, lo haré.' },
    {
      type: 'Answer (-)',
      text: "No, I won't.",
      translation: 'No, no lo haré.',
    },
  ]}
/>

<ImportantNote>
  Recuerda que también existe 'going to' para planes ya decididos.
</ImportantNote>
