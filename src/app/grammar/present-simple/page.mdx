# Present Simple

Se usa para rutinas, hechos generales y estados permanentes.

<GrammarExample
  structure="Subject + Verb (s/es for 3rd person) + Complement"
  examples={[
    {
      type: '1st Person (Affirmative)',
      text: 'I study English every day.',
      translation: 'Estudio inglés todos los días.',
    },
    {
      type: '3rd Person (Affirmative)',
      text: 'She studies English every day.',
      translation: 'Ella estudia inglés todos los días.',
    },
    {
      type: 'Negative',
      text: "I do not (don't) study English.",
      translation: 'No estudio inglés.',
    },
    {
      type: '3rd Person Negative',
      text: "He does not (doesn't) study English.",
      translation: 'Él no estudia inglés.',
    },
    {
      type: 'Interrogative',
      text: 'Do you study English?',
      translation: '¿Estudias inglés?',
    },
    { type: 'Answer (+)', text: 'Yes, I do.', translation: 'Sí.' },
    { type: 'Answer (-)', text: "No, I don't.", translation: 'No.' },
  ]}
/>

<ImportantNote>
  ¡Recuerda agregar 's' o 'es' al verbo cuando hables de He, She o It en
  afirmativo!
</ImportantNote>
