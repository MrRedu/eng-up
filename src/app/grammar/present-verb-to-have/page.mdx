# Present Simple: Verb "to have"

El verbo "to have" se usa para posesión, características y también como auxiliar o en modismos.

<GrammarExample 
  structure="Subject + have/has + Complement"
  examples={[
    { type: "Affirmative (I/You/We/They)", text: "I have a big house.", translation: "Tengo una casa grande." },
    { type: "Affirmative (He/She/It)", text: "He has a new car.", translation: "Él tiene un carro nuevo." },
    { type: "Negative", text: "I don't have enough time.", translation: "No tengo suficiente tiempo." },
    { type: "Negative (3rd Person)", text: "She doesn't have a cat.", translation: "Ella no tiene un gato." }
  ]}
/>

<ImportantNote>
  Recuerda que en negativo e interrogativo usamos el auxiliar **do/does** y el verbo vuelve a ser **have** siempre.
</ImportantNote>

## Como Modismo (Have as an idiom)

A veces "have" no significa "tener", sino que forma parte de una expresión:
- **Have lunch:** Almorzar.
- **Have a party:** Hacer una fiesta.
- **Have fun:** Divertirse.
- **Have a talk:** Conversar.
- **Have a rest:** Descansar.
