# Past Simple

<Typography variant="p" className="text-center">
  {`Se utiliza para acciones que terminaron en un momento específico del pasado.`}
</Typography>

<br />
<br />

<GrammarExample
  structure="Subject + Verb(-ed / irregular) + Complement"
  examples={[
    {
      type: 'Affirmative',
      text: 'I studied English yesterday.',
      translation: 'Estudié inglés ayer.',
    },
    {
      type: '3rd Person',
      text: 'He went to the park.',
      translation: 'Él fue al parque.',
    },
    {
      type: 'Negative',
      text: "I did not (didn't) study English.",
      translation: 'No estudié inglés.',
    },
    {
      type: 'Interrogative',
      text: 'Did you study English?',
      translation: '¿Estudiaste inglés?',
    },
    { type: 'Answer (+)', text: 'Yes, I did.', translation: 'Sí, lo hice.' },
    {
      type: 'Answer (-)',
      text: "No, I didn't.",
      translation: 'No, no lo hice.',
    },
  ]}
/>

<Alert variant="info" title="Dato importante">
  Cuando usamos el auxiliar **'did'**, el verbo vuelve a su forma base (ej: **'Did
  you go?'** en lugar de **'Did you went?'**).
</Alert>
