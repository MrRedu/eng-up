# Future Continuous

Se usa para acciones que estarán en progreso en un momento específico del futuro.

<GrammarExample
  structure="Subject + will be + Verb-ing + Complement"
  examples={[
    {
      type: 'Affirmative',
      text: 'I will be studying at 10 PM.',
      translation: 'Estaré estudiando a las 10 PM.',
    },
    {
      type: 'Negative',
      text: "They won't be working tomorrow.",
      translation: 'Ellos no estarán trabajando mañana.',
    },
    {
      type: 'Interrogative',
      text: 'Will you be sleeping?',
      translation: '¿Estarás durmiendo?',
    },
    { type: 'Answer (+)', text: 'Yes, I will.', translation: 'Sí, lo estaré.' },
    {
      type: 'Answer (-)',
      text: "No, I won't.",
      translation: 'No, no lo estaré.',
    },
  ]}
/>

<ImportantNote>Usamos 'will be' para todos los sujetos.</ImportantNote>
