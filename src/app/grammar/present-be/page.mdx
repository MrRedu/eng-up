# Present Simple: Verb "to be"

<Alert variant="info" title="Dato importante">
  Recuerda las contracciones comunes: **I'm**, **You're**, **He's**, **She's**, **It's**, **We're**, **They're**. En negativo puedes usar **isn't** y **aren't**.
</Alert>
