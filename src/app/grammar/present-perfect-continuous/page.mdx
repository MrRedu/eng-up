# Present Perfect Continuous

Se usa para acciones que comenzaron en el pasado y continúan en el presente, o para hablar de cuánto tiempo ha estado ocurriendo algo.

  structure="Subject + have/has + been + Verb-ing + Complement"

    { type: "Negative", text: "It hasn't been raining today.", translation: "No ha estado lloviendo hoy." },
    { type: "Interrogative", text: "How long have you been waiting?", translation: "¿Cuánto tiempo has estado esperando?" }

## For vs. Since

- **For:** Se usa para un periodo de tiempo (*for two hours, for five days*).
- **Since:** Se usa para un punto específico en el tiempo (*since 9:00, since Monday, since 2010*).

<ImportantNote>
  A diferencia del Present Perfect, aquí el énfasis está en la **duración** y la **continuidad** de la acción.
</ImportantNote>
