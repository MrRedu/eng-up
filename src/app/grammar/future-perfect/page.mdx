# Future Perfect

Se usa para acciones que habrán terminado en un punto específico del futuro.

<GrammarExample
  structure="Subject + will have + Past Participle + Complement"
  examples={[
    {
      type: 'Affirmative',
      text: 'I will have finished my course by June.',
      translation: 'Habré terminado mi curso para junio.',
    },
    {
      type: 'Negative',
      text: "She won't have arrive by then.",
      translation: 'Ella no habrá llegado para entonces.',
    },
    {
      type: 'Interrogative',
      text: 'Will you have done it by tomorrow?',
      translation: '¿Lo habrás hecho para mañana?',
    },
    {
      type: 'Answer (+)',
      text: 'Yes, I will.',
      translation: 'Sí, lo habré hecho.',
    },
    {
      type: 'Answer (-)',
      text: "No, I won't.",
      translation: 'No, no lo habré hecho.',
    },
  ]}
/>

<ImportantNote>
  A menudo se usa con la expresión 'by' (para tal momento).
</ImportantNote>
